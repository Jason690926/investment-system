# 優化2 — 強勢突破追蹤備案：設計

- 日期：2026-05-22
- 觸發：用戶 5/21 報表，東捷（long）操作框架寫「現價 120.5 已超出進場區，
  不宜現價追多，等回測 103.5~101.25」，5/22 卻直接漲停 +9.96%（站上 127.5
  壓力）。波段框架「不追高」紀律對「放量突破前高」的強勢股偏保守，連續多檔
  強勢突破股（東捷/合晶/臻鼎）被「等回測」攔在門外。
- 對應 plan：§二十八（待寫）

---

## 一、問題陳述

`analyze_market_only` 五、操作框架 long 模板只有 3 個 bullet：
進場價（= entry_zone 回測區）/ 停損 / 目標。當現價已**放量突破** swing
`range_high`（前高），entry_zone 回測區已在下方很遠 → AI 只能說「等回測、
不追高」。

李佛摩原則「沒突破不追價」的反面是：**突破且量能確認時，追進是紀律內的動作**，
不是追高。現行模板缺這個分支 → 強勢突破股一律被判「等回測」，錯失波段啟動。

## 二、設計原則（不破壞既有紀律）

- 預設仍是「不追高」：現價在 entry_zone 上方但**未帶量突破前高** → 維持
  「等回測」，不變。
- 只有「現價站上 range_high 且當日量 ≥ 突破量門檻」這個**程式可驗證**的
  強勢狀態成立時，才解鎖「追進」選項。追進 ≠ 追高：前者有量能確認、有明確
  停損（前高），後者沒有。

## 三、影響範圍

- `modules/ai_analyzer_v2.py`：long 操作框架模板（line ~1104）+ 一個程式
  計算的「強勢突破狀態」注入。
- 可能復用既有「突破量門檻」（vol_5avg×1.5）與 swing `range_high`。

## 四、修法設計

### 決策 1：程式計算「強勢突破狀態」（不交給 AI 判斷）

新增 helper（或併入既有 dynamic_block 組裝）：
```
強勢突破 = (現價 > swing.range_high) AND (今日量 ≥ 突破量門檻)
```
- `range_high` 來自 `calc_swing_levels`（已有）。
- 今日量 / 突破量門檻已是既有注入值。
- 結果注入 prompt：`【強勢突破狀態】成立 / 未成立`（程式計算，禁 AI 推翻）。

### 決策 2：long 操作框架模板加條件式「追進」bullet

long 模板（line 1104-1109）改為依【強勢突破狀態】二擇一：

- **未成立**（預設）：維持現行 3 bullet（進場價=回測區/停損/目標）。
- **成立**：進場價 bullet 改為（或追加）「強勢突破追蹤」：
  ```
  ▶ 強勢突破追蹤：現價已放量站上前高 [range_high] 元（量達突破門檻），
    可順勢追進；追進停損 = [range_high] 元（跌回前高即假突破）
  ▶ 回測進場（保守）：[entry_zone] 元 — 若不追高，待回測此區再進
  ▶ 目標：[target] 元（P&F 等幅量度）
  ```
  → 同時給「追進」與「回測進場」兩條路，讓讀者依風險偏好選，而非只有「等回測」。

### 決策 3：維持 schema 鐵律

仍受 line 1102「long 用進場/停損/目標術語」鐵律約束；「強勢突破追蹤」是
進場 bullet 的強勢變體，停損/目標 bullet 不變。short/neutral 模板不動。

## 五、測試計畫

- 「強勢突破狀態」helper 為純函式 → 單測：現價>range_high+量達標→成立；
  現價>range_high 但量不足→未成立；現價≤range_high→未成立。
- prompt 組裝：成立時 long 模板含「強勢突破追蹤」字串。
- 既有測試零退化。

## 六、風險與回滾

- 純加性：新增程式旗標 + long 模板條件分支；short/neutral/未突破情境
  byte-identical。單 commit 可 revert。
- ⚠️ prompt 行為改動，需 deploy + 重跑驗證 AI 是否在強勢股輸出追進選項、
  在非強勢股維持「等回測」。
- 殘留取捨：追進本質風險高於回測進場；模板會明列「追進停損=前高」控制風險，
  並保留「回測進場（保守）」並陳，由讀者選擇。

## 七、驗收

1. pytest 全綠（新 helper 測試）+ py_compile。
2. Deploy 後重跑：強勢突破股（如東捷站上 127.5+量達標）操作框架出現
   「強勢突破追蹤」追進選項 + 並陳「回測進場（保守）」；非突破的 long 股
   維持原 3 bullet「等回測」不變。

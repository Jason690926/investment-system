> 最後更新：2026-04-27

## 一、系統概述

自動化投資建議系統，每天從網路抓取全球財經資料，透過 Claude AI 分析後產生 HTML 報表並寄送 Email。

### 核心功能
- 全球股市、美債、黃金、石油多時間維度數據
- 台股大盤技術面分析
- 持股追蹤個別 AI 分析（含量能、K線、條件式操作建議）
- 李佛摩法則訊號（含進場區間、目標價、停損）
- 產業投資方向建議
- 自動判斷日報（週一至週五）/ 週報（週末/假日）

## 八、各模組詳細說明

### 8-1. data_fetcher.py — 資料抓取

**設計原則：**
- 所有來源同時平行抓取（ThreadPoolExecutor）
- 使用 `end + timedelta(days=1)` 確保包含今日資料
- 台股抓取有重試機制（失敗後等 2 秒再試一次）

**多時間維度漲跌幅：**

| 資產類型 | 時間維度 |
|---------|---------|
| 全球股市 | 今日、7日、30日、60日 |
| 宏觀資產（美債/黃金/石油） | 今日、7日、14日、30日、60日 |
| 台股持股 | 今日、5日、20日 |

**量能資料（台股）：**
- 今日成交量 / 昨日成交量 / 5日平均成交量
- 今日/昨日量比 → 自動判斷爆量/放量/縮量/急縮量
- 今日/均量量比

### 8-2. technical.py — 技術分析

**計算指標：**

| 指標 | 說明 |
|------|------|
| MA5 / MA20 / MA60 | 移動平均線 |
| RSI(14) | 相對強弱指標 |
| 支撐一 | 近20日最低價 |
| 支撐二 | 近10日最低價 |
| 壓力一 | 近10日最高價 |
| 壓力二 | 近20日最高價 |
| 進場區間 | 現價 ±0.5% |
| 目標一 | 近20日高點 |
| 目標二 | 斐波那契 1.618 延伸 |
| 停損 | max(現價 - ATR×2, 支撐×0.99) |

**趨勢判斷邏輯：**
```
資料 ≥ 60天：MA5 > MA20 > MA60 → 強勢多頭
              MA5 > MA20         → 多頭
              MA5 < MA20 < MA60 → 強勢空頭
              MA5 < MA20         → 空頭
              其他               → 盤整
資料 ≥ 20天：用 MA5 / MA20 判斷
資料 ≥ 5天： 用 MA5 判斷
資料不足：   回傳「資料不足」
```

### 8-3. candlestick.py — K線型態偵測

**已修正 Bug 清單：**

| Bug | 修正說明 |
|-----|---------|
| 跳空誤判大陽線/大陰線 | 跳空優先判斷，有跳空時不顯示大陽線/大陰線 |
| 射擊之星 + 倒錘子線衝突 | 依漲跌互斥：陰線→射擊之星，陽線→倒錘子線 |
| 三白兵條件太嚴 | 放寬為只要收盤遞增即可 |
| 十字星與墓碑十字重疊 | 十字星排除上影線極長的情況 |

**型態清單：**

| 型態 | 類型 | 說明 |
|------|------|------|
| 跳空大漲 | 看多 | 今日最低 > 昨日最高，收陽 |
| 跳空大跌 | 看空 | 今日最高 < 昨日最低，收陰 |
| 跳空高開拉回 | 中性 | 跳空高開後收低 |
| 跳空低開反彈 | 中性 | 跳空低開後收高 |
| 錘子線 | 看多 | 長下影線陽線 |
| 吊人線 | 看空 | 長下影線陰線（高位） |
| 倒錘子線 | 看多 | 長上影線陽線 |
| 射擊之星 | 看空 | 長上影線陰線 |
| 墓碑十字 | 看空 | 開收幾乎在低點 |
| 十字星 | 中性 | 開收幾乎相同（排除墓碑十字） |
| 大陽線 | 看多 | 實體 > 70%（非跳空） |
| 大陰線 | 看空 | 實體 > 70%（非跳空） |
| 多頭吞噬 | 看多 | 今陽完全吞昨陰 |
| 空頭吞噬 | 看空 | 今陰完全吞昨陽 |
| 烏雲蓋頂 | 看空 | 酒田：跳空高開後收低過前日中線 |
| 曙光初現 | 看多 | 酒田：跳空低開後收高過前日中線 |
| 三白兵 | 看多 | 連三陽，收盤遞增 |
| 三黑鴉 | 看空 | 連三陰，收盤遞減 |
| 早晨之星 | 看多 | 酒田：陰+小實體+大陽 |
| 黃昏之星 | 看空 | 酒田：陽+小實體+大陰 |
| 三山頂 | 看空 | 酒田：三次高點無法突破 |
| 三川底 | 看多 | 酒田：三次低點止跌 |

### 8-4. ai_analyzer.py — Claude AI 分析

**模型版本：** `claude-sonnet-4-6`
（2026-04-27 更新，舊版 claude-sonnet-4-5 將於 2026-06-15 停用）

**數字驗證機制（所有函式）：**
- 所有價位只能使用程式提供的真實數字
- 嚴禁自行描述歷史起始價格（如「從60元漲到100元」）
- 只能用百分比描述走勢（如「近30日漲幅+XX%」）
- 7個分析函式均已加入此規定

**分析函式說明：**

| 函式 | 說明 | max_tokens |
|------|------|-----------|
| `analyze_global_market` | 全球市場分析，含多時間維度大格局解讀 | 2000 |
| `analyze_taiwan_market` | 台股大盤技術面 + 操作方向（不推薦個股） | 2000 |
| `analyze_macro_assets` | 美債/黃金/石油白話解析 | 2000 |
| `analyze_watchlist_stock` | 個股完整分析（五項結構） | 1500 |
| `analyze_weekly_watchlist` | 週報個股分析（五項結構） | 1500 |
| `analyze_weekly_global` | 週報全球市場回顧 + 下週展望 | 2000 |
| `analyze_weekly_taiwan` | 週報台股回顧 + 下週操作建議（不推薦個股） | 2000 |
| `get_sector_recommendations` | 產業投資方向（只給方向，不推薦個股） | 2000 |

**持股分析五項結構：**
```
一、今日走勢 + K線型態解讀
    結合1日/5日/20日背景，量能爆量/放量/縮量/持平
二、明日操作策略（條件式）
    如果開盤站穩 XXX 元：...
    如果無法突破 XXX 元：...
    如果跌破 XXX 元：...
三、明日關鍵價位（真實數字，不可修改）
四、持倉者建議（加碼/減碼/續抱/停利）
五、特別注意事項（明日需觀察的關鍵訊號）
```

**API 呼叫設定：**
- 批次大小：5支 / 批次間隔：2秒 / Timeout：90秒
- 自動重試：rate limit 等15秒，其他等5秒，最多1次

**AI 大格局分析原則：**
- 今日漲跌必須放在7/30/60日趨勢脈絡下解讀
- 禁止因單日小幅波動使用「恐慌」「崩跌」等誇大詞彙
